# Simon Says Game 🎮

A simple Simon Says Game built using HTML, CSS, and JavaScript.

## Features
- Random color sequence generation
- Score tracking
- High score tracking
- Game-over animation
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Press any key to start.
2. Watch the flashing button.
3. Repeat the sequence by clicking the buttons.
4. The sequence gets longer every level.
